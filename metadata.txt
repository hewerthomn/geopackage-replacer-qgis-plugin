[about]
Geopackage Replacer updates any .gpkg used in the current project. Pick the new .gpkg (or .zip) and the origin from the project. The plugin validates and previews layers, feature counts and SRIDs before replacing, creates a timestamped backup and (optionally) reloads the project.
